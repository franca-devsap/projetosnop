# /fiori-pattern-update

1. Detect whether the rule is customer-only or cross-customer.
2. Write it to the right target:
   - customer `docs/CONVENTIONS.md`
   - customer `docs/DECISIONS.md`
   - root `docs/SHARED_CONVENTIONS.md`
3. Append a dated line to the matching `LESSONS_LEARNED.md`.
